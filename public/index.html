Tracking for Wekinator

Select a tracking mode to begin. Each mode sends different OSC data to Wekinator for sound control.

Face Tracking

Track facial features including eyes, nose, and mouth positions. Sends 12 parameters to Wekinator.

Start Face Tracking

Body Tracking

Track body pose with 17 keypoints including shoulders, elbows, wrists, knees, etc. Sends 34 parameters to Wekinator.

Start Body Tracking

Setup Information

For Face Tracking, configure Wekinator with OSC address /wek/inputs/face and 12 inputs.

For Body Tracking, configure Wekinator with OSC address /wek/inputs/body and 34 inputs.

Both modes send OSC data to port 6448 on localhost (127.0.0.1).
